Mail Menu

  • Delete Selected Email
    • Delete the selected email
    • If no email has been selected, an alert dialog will pop up
  • New Email
    • Reset the form to let the user enter new recipients and allow him to Save or Send the email
    • If he chooses to Save the email, that email will go to the Draft folder
  • Reload
    • Reload the Inbox folder to get the received emails

Folder Menu

  • Add Folder
    • Add a new folder
    • If the provided name is empty or is Inbox, Sent or Draft (case-insensitive), an alert dialog will pop up
  • Delete Selected Folder
    • Delete the selected folder
    • If no folder has been selected, an alert dialog will pop up
  • Rename Selected Folder
    • Rename the selected folder
    • Only custom folders can be renamed
    • If the user wants to rename the Inbox, Sent or Draft folder, an alert dialog will pop up

Edit Menu

  • Edit Mail Configuration
    • Allow the user to modify and save the fields in the Mail Config Form

Help Menu

  • About
    • The Help (this) page that specifies how to use the Email App

The Form - Emails Outside Draft

  • Emails outside the Draft folder have no access to the Save and Send buttons.
  • A reply button is enabled for emails outside the Draft folder. When pressed, it allows the user to Save and Send the email.
    • When Save is pressed, it places the email in the Draft folder.
    • When Send is pressed, it sends the email to the provided recipients and store it in the database.
  • If there are regular attachments to an email outside the Draft folder, the user can save them to disk by clicking on the download icon.
  • When nothing is pressed while composing a new email, that email will not be saved.

The Form - Emails In Draft

  • The user can Save a draft email and Send it also.
    • When Save is pressed, it updates the draft email.
    • When Send is pressed, it sends the email and place the email in the Sent folder.
  • He cannot reply to the email.
  • He can add, remove attachments.

When Composing An Email

  • Recipients can be added by clicking the + button near the To, CC and BCC labels.
  • Attachments can be added by clicking the + button near the paperclip icon.
  • The user can always decide to remove the attachments by clicking on the erase icon on the right. This will remove the attachment added in the root of the project.
